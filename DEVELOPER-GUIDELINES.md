# SECCIÓN A: GREEN IT 
### Respecto al frontend:
- Para optimizar el tiempo de carga y el consumo de la página web, todas las imágenes utilizadas se comprimirán de antemano y se transformen a formato WebP.
- Se utilizará caché inteligente y lazy loading para reducir drásticamente los tiempos de carga.
- Se revisará el código de manera regular (anualmente) para optimizarlo, eliminando dependencias innecesarias y código muerto.
### Respecto al backend: 
- Se hará uso de la IA para acelerar los flujos de trabajo y optimizar consultas a la base de datos.
- Se realizarán pruebas de estrés trimestralmente para identificar y resolver problemas de rendimiento.
- Se hará uso de clases planas para aquellas consultas específicas que solo requieran de algunos campos, para evitar consultas grandes que recuperan datos innecesarios.

# SECCIÓN B: DISEÑO ÉTICO Y ACCESIBILIDAD
### Respecto a la accesibilidad:
- Se seguirán de cerca los requerimientos de la WCAG (Web Content Accessibility Guidelines).
- Se hará uso de herramientas automatizadas para mejorar la accesibilidad (Lighthouse, TAW...).
- Se trabajará con asociaciones dedicadas a la mejora de la accesibilidad online para evitar los sesgos e identificar áreas comúnmente ignoradas en el sector de la accesibilidad.
### Respecto a la ética de datos:
- No se pedirá a los usuarios más que los datos estrictamente necesarios (nombre, apellidos y dirección en caso de envíos, DNI en caso de recogida en tienda).
- Los datos se eliminarán tras la compra, en caso de usuarios no registrados. En el caso de los usuarios registrados, los datos se borrarán de la base de datos cuando se elimine la cuenta.
### En el caso de la UX ética:
- La navegación será transparente: el usuario debe saber en todo momento lo que está haciendo, qué datos se están recogiendo, etc.
- Si el usuario cierra un anuncio o un pop-up, este se mantendrá cerrado sin que otro anuncio lo sustituya.
- Las políticas de privacidad y de cookies serán fácilmente accesibles en el footer, con un tamaño y contraste que faciliten su visibilidad. Estarán explicadas de forma clara.
